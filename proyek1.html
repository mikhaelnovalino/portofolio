Aplikasi Penjualan Tiket Konser

Saya telah berhasil membuat Aplikasi untuk penjualan tiket konser Tour Neck Deep yang akan diselenggarakan di beberapa kota dan negara di Asia menggunakan bahasa pemrograman Java pada aplikasi NetBeans. Program tersebut telah berhasil berjalan dengan lancar tanpa bug.

Berikut beberapa keunggulan proyek ini:

  • Desain responsif
  • Mudah diakses
  • Bermanfaat bagi penyelenggara konser

Teknologi: Java dan NetBeans

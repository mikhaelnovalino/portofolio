Intuisi Kopi Online Website

Saya telah berhasil membuat website toko kopi yang bisa diakses secara online untuk membantu pengguna dalam memesan kopi yang mereka suka tanpa perlu membuang waktu pergi ke coffee shop. Website ini saya buat dengan menggunakan bahasa pemrograman HTML dan style CSS yang telah disempurnakan dengan tambahan JavaScript untuk tampilan yang lebih menarik dan responsif.

Berikut beberapa keunggulan proyek ini:

  • Desain responsif
  • Mudah diakses
  • Bermanfaat bagi UMKM

Teknologi: HTML, CSS, dan JavaScript
